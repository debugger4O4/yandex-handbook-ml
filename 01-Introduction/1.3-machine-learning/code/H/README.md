### Предсказание по типам заведений

В этой задаче вам нужно будет загрузить py-файл, в котором реализован класс RubricCityMedianClassifier.

Обратите внимание, что в вашем файле должны быть импортированы все необходимые библиотеки.

Гарантируется, что в данных, на которых будет тестироваться ваша модель, уже будет столбец modified_rubrics, причём в 
тестовой выборке не будет значений, которых не было в обучающей.
